    06 Interface compare Comparator

    Comparator与Comparable的对比¶

    Comparable 接口让一个类自身具备比较能力。类实现 Comparable 接口，并实现 compareTo 方法，这样这个类的实例可以根据自身的某个特征进行比较。

    Q：但是如果我把comparable的implementation改成名字对比，不是也能实现名字的排序吗？

    A：是的，你可以将 Comparable 的 compareTo 实现改为按名字比较，这样就可以实现按名字排序。但是这样做会改变 Dog 类的自然顺序，使得自然顺序变成按名字排序，而不是按大小排序。这会限制你的排序方式，因为每个类只能有一个自然顺序。

    当你需要添加新的排序方式时，只需创建一个新的比较器类，而不需要修改 Dog 类中的 compareTo 方法。

    An object that implements Comparable can compare another object to itself, whereas a Comparator compares two objects other than itself.

    A good way to remember this is that a Comparable has an inherent property of being able to be compared, while a Comparator is an external source of truth.

    在Comparable的示例基础上的举例说明Comparator的区别¶

    import java.util.Comparator;

public class Main {
    public static void main(String[] args) {
        Dog[] arr = new Dog[3];
        arr[0] = new Dog("gaga", 5);
        arr[1] = new Dog("bubu", 10);
        arr[2] = new Dog("kiki", 15);
        Dog x = (Dog) MAX(arr);
        System.out.println(x.getName());

        Person[] persons = new Person[3];
        persons[0] = new Person("John", 180);
        persons[1] = new Person("Jeff", 179);
        persons[2] = new Person("Jason", 175);
        Person maxPerson = (Person) MAX(persons);
        System.out.println(maxPerson.getName());

        //comparator
        Dog.NameComparator compareDogName = new Dog.NameComparator();
        System.out.println(compareDogName.compare(arr[0], arr[1]));

    }

    public static OurComparable MAX(OurComparable[] items) {
        if (items == null || items.length == 0) {
            return null;
        }
        OurComparable maxItem = items[0];
        for(OurComparable item: items){
            if(item.compareTo(maxItem)>0){
                maxItem = item;
            }
        }
        return maxItem;
    }
}

class Dog implements OurComparable{
    private String name;
    private int age;
    public Dog(String name , int age){
        this.name = name;
        this.age = age;
    }
    public String getName(){
        return name;
    }

    @Override
    public int compareTo(Object o){
        if (o instanceof Dog otherDog) {
            return Integer.compare(this.age, otherDog.age);
        }
        throw new IllegalArgumentException("Cannot compare Dog with non-Dog object");
    }

    public static class NameComparator implements Comparator<Dog> {
        public int compare(Dog a, Dog b){
            return(a.getName().compareTo(b.getName()));
        }
    }

}

class Person implements OurComparable{
    private String name;
    private int height;
    public Person(String name , int height){
        this.name = name;
        this.height = height;
    }
    public String getName(){
        return name;
    }

    @Override
    public int compareTo(Object o){
        if (o instanceof Person otherPerson) {
            return Integer.compare(this.height, otherPerson.height);
        }
        throw new IllegalArgumentException("Cannot compare Dog with non-Dog object");
    }
}
