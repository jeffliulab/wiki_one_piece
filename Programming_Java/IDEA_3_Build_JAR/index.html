    IDEA 3 Build JAR

    在 IntelliJ IDEA 中生成可执行的 Java 文件（通常是 JAR 文件）可以按照以下步骤进行。这个过程包含编译代码、打包 JAR 文件，并确保它包含一个清单文件，以指定程序的入口点（主类）。

    第一步：编译项目

    1. 打开 IntelliJ IDEA 并打开你的项目。
    2. 确保项目能成功编译：
    3. 在菜单栏中，选择 Build > Build Project（快捷键 Command + F9）。
    4. 如果编译成功，你会在 IDE 的底部看到 "Build completed successfully"。

    第二步：配置 Artifacts

    1. 打开 Project Structure 窗口：
    2. 在菜单栏中，选择 File > Project Structure（快捷键 Command + ;）。
    3. 配置 Artifacts：
    4. 在 Project Structure 窗口中，选择左侧的 Artifacts。
    5. 点击 + 按钮，然后选择 JAR > From modules with dependencies...。
    6. 在弹出的对话框中，选择你的主模块，然后选择主类（包含 main 方法的类）。
    7. 确保选中 "Extract to the target JAR" 选项，然后点击 OK。
    8. 确认配置：
    9. 你会看到一个新的 Artifact 被添加到了列表中。你可以双击它以确认输出目录和包含的文件。

    第三步：生成 JAR 文件

    1. 生成 JAR 文件：
    2. 在菜单栏中，选择 Build > Build Artifacts，然后选择刚才配置的 Artifact，并选择 Build。
    3. IDEA 会生成 JAR 文件，通常在 out/artifacts/<artifact_name>/ 目录中。

    第四步：运行 JAR 文件

    1. 打开终端：
    2. 在你的 Mac 上，打开 终端 应用。
    3. 导航到 JAR 文件所在目录：
    4. 使用 cd 命令导航到生成的 JAR 文件所在的目录，例如： bash cd path/to/your/project/out/artifacts/your_artifact_name/
    5. 运行 JAR 文件：
    6. 使用以下命令运行 JAR 文件： bash java -jar your_artifact_name.jar

    示例

    假设你的项目名为 MyProject，主类为 com.example.Main，生成的 JAR 文件名为 MyProject.jar。你的命令可能如下所示：

    ```bash cd /Users/yourusername/IdeaProjects/MyProject/out/artifacts/MyProject/ java -jar MyProject.jar
