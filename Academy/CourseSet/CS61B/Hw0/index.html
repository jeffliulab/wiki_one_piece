JavaExercises.java has 4 different methods for you to complete:

makeDice: This method takes returns a new array of integers [1, 2, 3, 4, 5, 6].
makeDice：这个方法需要返回一个新的整数数组，包含数字 [1, 2, 3, 4, 5, 6]。

takeOrder: This method takes in a String and returns a new array containing the orders of the customer. If the customer is Ergun, you should return an array of Strings ["beyti", "pizza", "hamburger", "tea"] in that order. If the customer is Erik, you should return an array of Strings ["sushi", "pasta", "avocado", "coffee"]. In any other case, return an empty String array of size 3.
NOTE: == behaves strangely with Strings for reasons we’ll see later in the course. You should check strings s1 and s2 for equality using s1.equals(s2) in Java.
takeOrder：这个方法接收一个字符串，并返回一个包含顾客订单的新数组。如果顾客是 "Ergun"，应返回 ["beyti", "pizza", "hamburger", "tea"]；如果顾客是 "Erik"，应返回 ["sushi", "pasta", "avocado", "coffee"]。对于其他任何顾客，应返回一个长度为3的空字符串数组。
注意：在比较字符串时，应该使用 s1.equals(s2) 而不是 ==。

findMinMax: This method takes an int[] array and returns the the positive difference between the maximum element and minimum element of the given array. You may assume the input array is nonempty.
findMinMax：这个方法接收一个整数数组，返回该数组中最大元素和最小元素之间的正差。可以假设输入数组是非空的。

hailstone: This method takes an int n as input and returns its hailstone sequence as a list of integers. The hailstone sequence is defined by the following procedure: pick a positive integer n as the start. If n is even, divide n by 2. If n is odd, multiply n by 3 and add 1. Continue this process until n is 1.
You should compute this using recursion with the provided helper method hailstoneHelper.
For this part, you can import List and ArrayList.
hailstone：这个方法接收一个整数 n 作为输入，返回其 hailstone 序列作为一个整数列表。hailstone 序列的定义如下：选择一个正整数 n 作为起点。如果 n 是偶数，将其除以 2；如果 n 是奇数，将其乘以 3 并加 1。继续这个过程直到 n 变为 1。需要使用递归来计算序列，并使用提供的辅助方法 hailstoneHelper。
 */

    public static int[] makeDice(){
        int[] x = new int[]{1,2,3,4,5,6};
        return x;
    }

    //takeOrder: This method takes in a String and returns a new array containing the orders of the customer.
    // If the customer is Ergun, you should return an array of Strings ["beyti", "pizza", "hamburger", "tea"] in that order.
    // If the customer is Erik, you should return an array of Strings ["sushi", "pasta", "avocado", "coffee"].
    // In any other case, return an empty String array of size 3.
    public static String[] takeOrder(String x){
        if (x.equals("Ergun")){
            return new String[]{"beyti", "pizza", "hamburger", "tea"};
        }
        else if (x.equals("Erik")){
            return new String[]{"sushi", "pasta", "avocado", "coffee"};
        }
        else {
            return new String[3];
        }

    }

    //findMinMax: This method takes an int[] array and returns  the positive difference between
    // //the maximum element and minimum element of the given array. You may assume the input array is nonempty.
    //@precondition: input array is nonempty
    public static int findMinMax(int[] arr){
        int min = arr[0];
        int max = arr[0];
        for(int i = 0; i < arr.length; i++){
            if (min > arr[i]){
                min = arr[i];
            }
            if(max < arr[i]){
                max = arr[i];
            }
        }
        return max - min ;
    }



}

    hailstone sequence

    import java.util.ArrayList;
public class HailstoneSequence {
    public static ArrayList<Integer> seq = new ArrayList<>();

    public static void main(String[] args){
        System.out.println("Hailstone Sequence");
        ArrayList<Integer> x = hailstone(6);
        for(int i = 0 ; i<x.size();i++){
            System.out.println(x.get(i));
        }
    }

//hailstone：这个方法接收一个整数 n 作为输入，返回其 hailstone 序列作为一个整数列表。
// hailstone 序列的定义如下：选择一个正整数 n 作为起点。如果 n 是偶数，将其除以 2；
// 如果 n 是奇数，将其乘以 3 并加 1。
// 继续这个过程直到 n 变为 1。需要使用递归来计算序列，并使用提供的辅助方法 hailstoneHelper。

    public static ArrayList<Integer> hailstone(int n){
        if(n==1){
            seq.add(1);
            return seq; //为什么这里必须return：因为n==1的情况下下面的n%2==1也能执行，所以在三个if的情况下这里必须return，否则就会栈溢出
        }
        if(n%2==0){
            seq.add(n);
            hailstone(n/2);
        }
        if(n%2==1){
            seq.add(n);
            hailstone(n*3+1);
        }

        //在Java中，如果一个方法声明了返回类型（如ArrayList<Integer>），则每个可能的执行路径都必须返回一个值。即使你认为所有的情况都已经被处理了，Java编译器还是需要明确知道每个可能的路径都有返回值。
        //在你的代码中，虽然你正确地处理了基准条件和递归条件，但编译器不能确定在所有情况下方法都一定会返回一个值。这就是为什么你需要确保在所有可能的路径上都有返回值。你可以通过在递归调用中添加返回语句来解决这个问题。

        return seq;

    }

}

    //sum: This method takes a list List<Integer> L and returns the total sum of the elements in that list.
    // If the list is empty, the method should return 0.
    public static int sum(ArrayList<Integer> L){
        int s = 0;
        for(int i = 0; i<L.size();i++){
            s += L.get(i);
        }
        return s;
    }

    //evens: This method takes a list List<Integer> L and returns a new list containing the even numbers of the given list.
    // If there are no even elements, it should return an empty list.
    public static ArrayList<Integer> evens(ArrayList<Integer> L){
        ArrayList<Integer> l = new ArrayList<Integer>();
        for(int i = 0 ; i < L.size();i++){
            if(L.get(i)%2==0){
                l.add(L.get(i));
            }
        }
        return l;
    }

    //        common: This method takes two lists List<Integer> L1,
    //        List<Integer> L2 and returns a new list containing the items present in both of the two given lists.
    //        If there are no common items, it should return an empty list.
    public static ArrayList<Integer> common(ArrayList<Integer> L1, ArrayList<Integer> L2){
        ArrayList<Integer> list = new ArrayList<Integer>();
        for(int i = 0 ; i < L1.size() ; i++){
            for(int j = 0 ; j < L2.size() ; j++){
                if(L1.get(i).equals(L2.get(j))){
                    list.add(L1.get(i));
                }
            }
        }
        return list;
    }

    //        countOccurrencesOfC: This method takes a list and a character List<String> words,
    //        char c and returns the number of occurrences of the given character in a list of strings.
    //        If the character does not occur in any of the words, it should return 0.
    public static int countOccurrencesOfC(ArrayList<String> words, char c){
        int n = 0;
        for(int i = 0 ; i<words.size();i++){
            for(int j = 0 ; j< words.get(i).length();j++){
                if(words.get(i).charAt(j) == c){
                    n++;
                }
            }
        }
        return n;
    }


}

//letterToNum: This method returns a map from every lower case letter to the number corresponding to its ordering in the alphabet,
// where ‘a’ corresponds to 1 and ‘z’ corresponds to 26.
    public static HashMap<String, Integer> letterToNum(){
        HashMap<String, Integer> maps = new HashMap<>();
        char x = 'a';
        for(int i = 1 ; i <= 26; i++){
            maps.put(String.valueOf((char)(x+i-1)),i);
        }
        return maps;
    }


//squares: This method takes a list List<Integer> nums and returns a map from the integers in the list to their squares.
//// If the given list is empty, it should return an empty map.
    public static HashMap<Integer, Integer> squares(List<Integer> list){
        HashMap<Integer, Integer> x = new HashMap<>();
        for(Integer num: list){
            x.put(num, num * num );
        }
        return x;
    }


//countWords: This method takes a list List<String> words and returns a map from words in the list
// to the number of times they appear. If the given list is empty, it should return an empty map.
//For this part, you can import TreeMap.
    public static TreeMap<String, Integer> countWords(List<String> list) {
        TreeMap<String, Integer> x = new TreeMap<>();

        for (int i = 0; i < list.size(); i++) {
            if(!x.containsKey(list.get(i))){
                x.put(list.get(i),1);
            }
            else{
                x.put(list.get(i), (x.get(list.get(i)))+1 );
            }

        }
        return x;
    }
}

public class Dessert{
    //Two instance variables: int flavor and int price.
    public int flavor;
    public int price;
    //One static variable int numDesserts that keeps track of the number of desserts created so far.
    public static int numDesserts = 0;

    //A constructor that takes two parameters int flavor and int price and sets the instance variables accordingly.
    public Dessert(int flavor, int price){
        this.flavor = flavor;
        this.price = price;
        numDesserts++;
    }

    //A method public void printDessert() that prints the flavor and price of the dessert,
    // along with the total number of desserts created so far, separated by a space.
//For example,
// if we create a dessert with flavor 1 and price 2, and then call its printDessert() method, it should print 1 2 1.
//If we then create a dessert with flavor 3 and price 4, and then call its printDessert() method, it should print 3 4 2.
    public void printDessert(){
        System.out.printf("%d %d %d%n", flavor, price, numDesserts);
    }

    //Lastly, a method public static void main(String[] args) that only prints the line I love dessert! when executed.
//Be sure to implement the above behavior exactly, otherwise you may not pass the tests!
    public static void main(String[] args) {
        System.out.println("I love dessert!");
        Dessert a1 = new Dessert(1,2);
        a1.printDessert();
        Dessert a2 = new Dessert(2,3);
        a2.printDessert();
    }

}
