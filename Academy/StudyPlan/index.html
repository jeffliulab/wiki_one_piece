    Tech Stack - 100%¶

    Fronted Development - 5%¶

    涉及用户界面和用户体验，使用HTML、CSS、JavaScript及其框架（如React、Angular、Vue.js）等技术来创建和设计网页的视觉效果和交互功能。

    Backend Development - 30%¶

    涉及服务器端逻辑、数据库和应用程序接口（API）的开发。常用的后端语言和框架包括Node.js、Python（Django、Flask）、Ruby on Rails、Java（Spring）、PHP（Laravel）等。

    Programming: Java, Python, JavaScript(Node.js) Framework: Spring Boot(Java), Django(Python), Express.js(Node.js) Database: PostgreSQL, MongoDB, Redis DevOps: Docker, Kubernetes, AWS(EC2, S3, Lambda), Jenkins Collaboration Tools: Git, Github

    Database - 5%¶

    设计和管理数据存储。常用的数据库包括关系型数据库（如MySQL、PostgreSQL）和非关系型数据库（如MongoDB、Redis）。

    该技术栈对后端开发最重要，在人工智能和机器人领域了解即可。

    Server and Deployment - 5%¶

    管理服务器、配置和维护服务器环境、部署应用程序。常用的工具和平台包括Docker、Kubernetes、AWS、Azure、Heroku等。

    Artificial Intelligence - 35%¶

    Programming: Python Framework: TensorFlow, PyTorch, Scikit-learn Data: NumPy, Pandas, SciPy NLP: Transformers (Hugging Face), SpaCy CV: OpenCV, TensorFlow/Keras（Keras现在集成在TensorFlow内） BigData: Apache Spark, Kafka Math: Linear, Probability, Statistic

    Robotics - 20%¶

    Programming: ROS, C++ Framework: ROS, Gazebo, MoveIt! CV: OpenCV, TensorFlow/Keras（Keras现在集成在TensorFlow内） Math: Linear, Control Theory, Dynamics(动力学)

    Study plan for Robotics(AI) Industry¶

    为了更具体地为机器人和AI领域的工作做准备，你需要学习和掌握以下技能和技术：

    编程语言¶

    • Python：广泛用于AI和机器学习领域，拥有丰富的库和框架，如TensorFlow、PyTorch、scikit-learn等。
    • C++：常用于机器人学中的低层次控制和性能关键部分。

    机器学习和深度学习¶

    • 数学基础：线性代数、概率论和统计学是理解机器学习算法的基础。
    • 算法和框架：学习常见的机器学习算法和框架，如TensorFlow、PyTorch、Keras等。

    计算机视觉¶

    • 图像处理基础：学习OpenCV等库，理解图像处理的基本概念。
    • 深度学习应用：使用深度学习技术进行图像分类、对象检测、图像分割等任务。

    机器人学¶

    • 控制理论：理解机器人控制的基本原理和算法。
    • 机器人操作系统（ROS）：ROS是机器人开发的标准平台，了解其基本使用和开发方法。

    数据科学¶

    • 数据预处理和分析：熟悉pandas、NumPy等数据处理库。
    • 数据可视化：学习matplotlib、Seaborn等可视化工具。

    自然语言处理（NLP）¶

    • NLP基础：理解文本处理的基本方法和技术，如分词、词性标注、语法解析等。
    • 深度学习在NLP中的应用：学习Transformer、BERT等前沿模型。

    学习路径建议¶

    1. 基础编程：强化Python和C++编程技能。
    2. 数学和算法：学习线性代数、概率论和统计学，以及基本的机器学习和深度学习算法。
    3. 专注领域：根据兴趣选择计算机视觉、自然语言处理或机器人学中的一个或多个方向深入学习。
    4. 实践项目：通过参与开源项目、课程项目或实习，积累实际经验。
    5. 进阶学习：学习和使用深度学习框架（如TensorFlow、PyTorch）和机器人操作系统（ROS）。

    阶段5：机器学习基础（3-4个月）¶

    5.1 监督学习¶

    • 内容：回归、分类、支持向量机、决策树。
    • 资源：
    • Coursera: Machine Learning by Andrew Ng
    • 《统计学习方法》 by 李航

    5.2 无监督学习¶

    • 内容：聚类、降维、PCA。
    • 资源：
    • Coursera: Unsupervised Machine Learning by Stanford University
    • 《机器学习》 by Tom Mitchell

    阶段6：深度学习基础（3-4个月）¶

    6.1 神经网络基础¶

    • 内容：感知器、多层感知器、反向传播算法。
    • 资源：
    • Coursera: Neural Networks and Deep Learning by Andrew Ng
    • 《深度学习》 by Ian Goodfellow, Yoshua Bengio, and Aaron Courville

    6.2 深度学习进阶¶

    • 内容：卷积神经网络、递归神经网络、深度学习的优化技术。
    • 资源：
    • Coursera: Deep Learning Specialization by Andrew Ng
    • Fast.ai: Practical Deep Learning for Coders

    总结¶

    这个详细的学习路径涵盖了从基础数学到高阶机器学习和深度学习的所有必要知识。你可以按照这个路径一步步学习，并根据自己的进度和理解情况适当调整每个阶段的学习时间。在学习过程中，确保不断实践，通过编程和项目来巩固所学知识。

    Foundations¶

    软件工程是关于软件开发的系统化、规范化和量化的学科，旨在提高软件开发的质量、效率和可维护性。其主要内容包括：

    测试方法：白盒测试、黑盒测试、灰盒测试。 自动化测试：单元测试、集成测试、回归测试。
